# react的开发环境搭建

查询所有npm配置 
```shell script
npm config list --json 
```

配置npm的镜像
```shell script
npm config set registry https://registry.npm.taobao.org/
```

npm清理缓存命令
```shell script
npm cache clean --force
```

安装包
```shell script
npm install --save-dev webpack@4.41.5
```

卸载指定包
```shell script
npm uninstall webpack
```
